Detailed Description

determines the offset between the timestamps from the EDVS and the computer

This is needed as we need to synchronize all events from different sources, like EDVS, keyboard input and information from robots and other devices. To do this calibration a flashing window is displayed on the laptop and the eDVS has to be pointed to it. The calibration is then started by pressing the key 'o'. For debugging purposes the events are displayed in a separate window, as well as the center of mass of the image.
