Detailed Description

Author
Michel Frising Provides a concrete implementation of the abstract RobotListener to receive the servo states and the bumber states parsed from the robot event stream. At the moment all loging capabilities are commented out, but Prof. Conradth from NST in Munich announced that in a feature release of the OmniRob firmware broadcasting capabilities, such as the eDVS event stream will be implemented, where this class will come in handy.
